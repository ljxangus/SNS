    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1>
<a name="user-content-kivy" class="anchor" href="#kivy" aria-hidden="true"><span class="octicon octicon-link"></span></a>Kivy</h1>

<h3>
<a name="user-content-snsapp" class="anchor" href="#snsapp" aria-hidden="true"><span class="octicon octicon-link"></span></a>SNSAPP</h3>

<p><em>This is a prototype kivy app based on SNSAPI.</em>
You can read cross-platform information in one app.</p>

<h3>
<a name="user-content-how-to-use-this-app" class="anchor" href="#how-to-use-this-app" aria-hidden="true"><span class="octicon octicon-link"></span></a>How to use this app?</h3>

<blockquote>
<p>Right now, I haven't packed the APP as APK, so the way to install the app is not that easy. But don't worry, it is still so simple for you to use it.</p>
</blockquote>

<ul class="task-list">
<li>Install kivy launcher in your phone/PC/iPhone</li>
</ul><blockquote>
<p><a href="https://play.google.com/store/apps/details?id=org.kivy.pygame">Install Link on Android APP store</a></p>
</blockquote>

<ul class="task-list">
<li>Put all the files download from github into your Kivy launcher location(i.e.<code>sdcard0/kivy</code>)
Tips: make sure that there is a file <code>android.txt</code> in the location <code>sdcard0/kivy/SNS/</code>
</li>
<li>Launch Kivy Launcher on your devices</li>
<li>Click <code>SNSAPP</code>
</li>
</ul><h3>
